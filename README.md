# Churn Prediction Modeling

This project focuses on predicting customer churn using Machine Learning techniques in Python.

## 📌 Project Overview

Customer churn prediction helps businesses identify customers who are likely to stop using their service.
This project includes:

* Data preprocessing
* Model training
* Model evaluation
* Making predictions using the trained model

---

## 📁 Files in this Repository

* `training.ipynb` – Model training and evaluation
* `prediction.ipynb` – Making predictions using the trained model
* `app.py` – Script to run the churn prediction model
* `requirements.txt` – Required Python libraries

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Jupyter Notebook

---

## 🚀 How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/yeripallivijay/Churn-Prediction-Modeling.git
   ```

2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Run the training notebook:

   ```bash
   jupyter notebook training.ipynb
   ```

4. Run the prediction script:

   ```bash
   streamlit run app.py
   ```

---

## 🎯 Goal

Build a machine learning model that can accurately predict whether a customer will churn or not.
